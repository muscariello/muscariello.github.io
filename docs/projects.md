# Ongoing Projects

## AGNTCY

Linux Foundation project launched in 2025. Building infrastructure for the Internet of Agents — providing agent discovery, identity, messaging, and observability that enables specialized AI agents to collaborate across platforms and organizational boundaries.

[agntcy.org →](https://agntcy.org)

---

## A2A Protocol

Linux Foundation project launched in 2025. An open protocol enabling secure and interoperable communication between AI agents across different platforms and frameworks.

[a2a-protocol.org →](https://a2a-protocol.org)

---

## FD.io Hybrid ICN

Linux Foundation project (2019-2025). High-performance implementation of ICN network and transport stacks.

[View Project →](https://wiki.fd.io/view/HICN)

---

## CICN

FD.io Linux Foundation project (2017-2019). Content-Centric Networking implementation based on CCNx 1.0, featuring a VPP forwarder plugin, socket-based forwarder, consumer/producer API, and applications for HTTP ABR video delivery.

[View Project →](https://wiki.fd.io/view/Cicn)

---

## ICN2020

EU-Japan research project (2016-2019). Advancing Information-Centric Networking towards real-world deployment through research, innovative applications, and global-scale experimentation with federated testbeds.

[View Project →](http://www.icn2020.org/)

---

## IRT SystemX

Network Architectures research at IRT SystemX (2014-2017). Research on future network architectures including Information-Centric Networking and software-defined networking.

[View Project →](https://www.irt-systemx.fr/)
